Exercise: Blink and Fade an LED

Objective

Make an LED fade from dark to full bright and back using the built in Arduino examples.

There are many times where we want to do more than just turn something on or off using the Arduino. To do this we use the function: analogWrite(pin, ouptut) (output can be any number from 0=off to 255=on). However, the Arduino can only ouput 0V or 5V, so there is no true analog output (there are other components that can do this). In order to get around this, the Arduino uses Pulse Width Modulation (PWM). You can think of this as turning the pin on and off really fast while varying the amount of time it is on vs. off. The larger the on-percentage, the brighter an LED would be or the faster a motor would turn (check out the comments section for some examples).

Be careful when choosing pins to use with PWM because not all of the Arduino Uno's pins can be used with the analogWrite function. Only the digital pins with the ~ next to the number can be used (3, 5, 6, 9, 10, 11).

Steps and observations

  1. Build the circuit below.
  2. Open the Fade sketch under File->Examples->01.Basics->Fade.
  3. Upload the sketch to your arduino and watch the result.
  4. Use the Ossillosope to watch the output from the Arduino Uno's pin 9.
  5. Experiment by changing the delay length and the fadeAmount variable.
  6. Try changing the delay and the fadeAmount variable to make the output look like the blink example
